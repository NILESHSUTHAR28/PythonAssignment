# Python Programs – Dictionary and List Operations
Task 1: Student Marks Lookup using Dictionary

Description
This program uses a Python dictionary to store student names and their corresponding marks.  
The user is asked to enter a student’s name, and the program checks whether the student exists in the dictionary.

Functionality
- A dictionary named `marks` stores student names as keys and marks as values
- User inputs a student name
- If the name exists in the dictionary:
- The student’s marks are displayed
- If the name does not exist:
- A message "Student not found" is shown

Task 2: List Slicing and Reversal

Description
This program demonstrates how to:
- Create a list of numbers
- Extract the first five elements using slicing
- Reverse the extracted elements

Functionality
- A list containing numbers from 1 to 10 is created
- The first five elements are extracted using list slicing
- The extracted list is reversed using slicing
- The original list, extracted list, and reversed list are displayed
